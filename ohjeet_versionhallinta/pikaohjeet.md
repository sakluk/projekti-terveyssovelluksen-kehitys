# Pikaohjeet

Anna komento/terminaali-ikkunassa seuraava komento:

```bash
git --help
```

Saat pikaohjeet. Tässä on suomennetut Git-ohjeet:

```
$ git --help
käyttö: git <komento> [<argumentit>]

Nämä ovat yleisiä Git-komentoja, joita käytetään erilaisissa tilanteissa:

aloita työalue (katso myös: git help tutorial)
   clone      Kloonaa tietovarasto uuteen hakemistoon
   init       Luo tyhjä Git-tietovarasto tai alusta olemassa oleva uudelleen

työskentele nykyisen muutoksen parissa (katso myös: git help everyday)
   add        Lisää tiedostojen sisältö indeksiin
   mv         Siirrä tai nimeä uudelleen tiedosto, hakemisto tai symbolinen linkki
   restore    Palauta työskentelypuun tiedostot
   rm         Poista tiedostot työskentelypuusta ja indeksistä

tutki historiaa ja tilaa (katso myös: git help revisions)
   bisect     Käytä binäärihakua löytääksesi tallennus (commit), joka toi virheen
   diff       Näytä muutokset tallennusten ja/tai työskentelypuun välillä
   grep       Tulosta kaavaa vastaavat rivit
   log        Näytä tallennuslokit
   show       Näytä erilaisia objekteja
   status     Näytä työskentelypuun tila

kasvata, merkitse ja muokkaa yhteistä historiaa
   backfill   Lataa puuttuvat objektit osittaisessa kloonissa
   branch     Listaa, luo tai poista haaroja
   commit     Tallenna muutokset tietovarastoon
   merge      Yhdistä kaksi tai useampi kehityshistoria
   rebase     Toteuta tallennukset (commit) uudelleen toisen perustan päälle
   reset      Palauta nykyinen HEAD määritettyyn tilaan
   switch     Vaihda haaroja
   tag        Luo, listaa, poista tai vahvista GPG:llä allekirjoitettu tägi-objekti

tee yhteistyötä (katso myös: git help workflows)
   fetch      Lataa objektit ja viitteet toisesta tietovarastosta
   pull       Nouda ja integroi toisesta tietovarastosta tai paikallisesta haarasta
   push       Päivitä etäviitteet ja niihin liittyvät objektit

'git help -a' ja 'git help -g' listaavat käytettävissä olevat alikomennot ja
konseptioppaita. Katso 'git help <komento>' tai 'git help <konsepti>'
lukeaksesi tietystä alikomennosta tai konseptista.
Katso 'git help git' saadaksesi yleiskuvan järjestelmästä.
```
